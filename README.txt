Project: Number Guessing Game

Features to Implement:

Generate a Random Number: The program generates a random number between a specified range.
Take User Guesses: Allow the user to input their guesses for the generated number.
Provide Feedback: Provide hints to the user if their guess is too high or too low.
Track Number of Attempts: Keep track of the number of attempts the user takes to guess the correct number.
Offer Replay: After the game ends, ask the user if they want to play again.
